## üñ•Ô∏è What is a Computer Network?

A **computer network** is a system where two or more devices (like computers, laptops, mobiles, printers, or servers) are connected together to **share data and resources**.

The main purpose of a network is:

* To **share resources** (like the internet, files, or printers)
    
* To **enable communication** (emails, chats, or video calls)
    

**Examples:**

* Home Wi-Fi (small local network)
    
* Office LAN (Local Area Network)
    
* The Internet (a huge Wide Area Network)
    

## üåê What is an IP Address?

An **IP Address (Internet Protocol Address)** is a unique number assigned to every device connected to a network.  
It works like an **identity card** that helps devices recognize and communicate with each other.

üß† **Example:**

* IPv4: `192.168.1.10`
    
* IPv6: `2400:cb00:2048:1::c629:d7a2`
    

## What are IPV4 and IPV6?

**IPv4 (Internet Protocol version 4):**

* The older version of IP.
    
* Uses 32-bit addresses.
    
* Example: `192.168.1.1`
    
* Can generate around **4 billion unique addresses**.
    

**IPv6 (Internet Protocol version 6):**

* The newer version.
    
* Uses 128-bit addresses.
    
* Example: `2400:cb00:2048:1::c629:d7a2`
    
* Provides **trillions of addresses** ‚Äî solving the IPv4 shortage problem.
    

## What is subnet (sub networking)?

**Subnetting** means dividing a large network into smaller, manageable parts called **subnets**.  
This helps improve performance, management, and security.

üß† **Example:**  
If an office network has 100 devices, you can divide them into 3 subnets:

* HR Department
    
* IT Department
    
* Finance Department
    

This way, one department‚Äôs traffic doesn‚Äôt interfere with another.

### What is public subnet and private subnet?

**Public Subnet:**

* Directly accessible from the Internet.
    
* Example: Web servers, public APIs.
    

**Private Subnet:**

* Not directly accessible from the Internet.
    
* Example: Databases, backend services.
    

### Advantages of creating subnet

* **Security:** Keeps sensitive data separate.
    
* **Privacy:** Controls who can access what.
    
* **Isolation:** One subnet issue won‚Äôt affect others.
    

## ‚òÅÔ∏è What is a VPC (Virtual Private Cloud)?

A **VPC (Virtual Private Cloud)** is a **virtual network inside the cloud** (like AWS, GCP, or Azure) where you can securely manage your resources.

In a VPC, you can define:

* IP address range
    
* Public and private subnets
    
* Routing rules
    
* Internet gateways
    

Basically, a VPC is your **own private, secure network in the cloud**.

## üìè What is a CIDR Range?

**CIDR (Classless Inter-Domain Routing)** defines how many IP addresses a network can have.

üß† **Examples:**

* `192.168.1.0/24` ‚Üí 256 IPs
    
* `10.0.0.0/16` ‚Üí 65,536 IPs
    

The number after ‚Äú/‚Äù shows how many bits are fixed ‚Äî fewer bits mean more available IPs.

## üîå What are Ports?

**Ports** are like doors through which data enters or leaves a device.  
Each port is assigned to a specific service.

üß† **Common Ports:**

* Port 80 ‚Üí HTTP (Websites)
    
* Port 443 ‚Üí HTTPS (Secure Websites)
    
* Port 22 ‚Üí SSH (Remote Login)
    
* Port 25 ‚Üí SMTP (Email)

## üß± What is the OSI Model?

The **OSI (Open Systems Interconnection) Model** explains **how data travels** through a network.  
It has **7 layers**, each responsible for a specific task.

1. **Physical** ‚Äì Hardware and cables
    
2. **Data Link** ‚Äì Data transfer between nearby devices
    
3. **Network** ‚Äì IP addressing and routing
    
4. **Transport** ‚Äì Reliable data delivery (TCP/UDP)
    
5. **Session** ‚Äì Manages connection between systems
    
6. **Presentation** ‚Äì Data format and encryption
    
7. **Application** ‚Äì Interfaces for users (browser, email, etc.)
    

## üåç Real-World Examples of Networking Concepts

    Your router assigns local IPs to connected devices ‚Äî just like a mini-network.

4. **Web Browsing:**  
    When you open a website, your device connects to a **public IP** using **port 443 (HTTPS)**.
    

## ‚ùì Frequently Asked Questions (FAQs)

**Q1. What‚Äôs the difference between an IP address and a MAC address?**  
‚Üí An IP address identifies a device on a network, while a MAC address identifies the **physical hardware** (like your Wi-Fi card).

**Q2. Why do we have both IPv4 and IPv6?**  
‚Üí IPv4 addresses are limited; IPv6 was created to provide a much larger pool of unique addresses.

**Q3. Can two devices have the same IP address?**  
‚Üí Not on the same network. If it happens, it causes an **IP conflict**, and one device won‚Äôt connect properly.

**Q4. What‚Äôs the main use of a subnet?**  
‚Üí To **organize, secure, and optimize** large networks by dividing them into smaller sections.
